## 1. 类型概述

### 1.1 业务场景

同学关系是教育阶段建立的重要关系，往往能维持终身。从小学到大学，从培训班到兴趣班，同学关系覆盖了人生的各个学习阶段。

**典型使用场景**：
- 记录各个求学阶段的同学
- 组织同学聚会（按学校、班级）
- 维护校友网络
- 追溯同学的职业发展

### 1.2 设计目标

1. **明确教育信息**：学校、年级、班级等关键信息
2. **时间段明确**：入学-毕业的明确时间范围
3. **关系分层**：区分同班、同级、同校等亲疏
4. **支持多阶段**：小学、初中、高中、大学、研究生等

### 1.3 关键特点

- ✓ 支持多时间段（可能在多个学校是同学）
- ✓ 要求填写时间段（求学时间通常很明确）
- ✓ 双向关系（互为同学）
- ✓ 中高优先级（优先级：68）
- ✓ 6个专属字段：学校名称、教育阶段、年级班级、同学类型、共同活动、毕业后联系状态

## 3. 字段详细说明

### 3.1 学校名称（school）

**字段名**：`school`
**类型**：`string`
**必填**：是

**说明**：
记录共同就读的学校名称，是同学关系最核心的标识。

**示例值**：
- "清华大学"
- "北京市第四中学"
- "上海外国语大学附属外国语学校"
- "Stanford University"
- "新东方英语培训学校"

**验证规则**：
- 最大长度：100字符
- 不能为空

**使用建议**：
- 使用学校全称
- 对于知名学校可以使用简称（如：清华、北大）
- 培训机构也可以记录（新东方、学而思等）

---

### 3.2 教育阶段（educationStage）

**字段名**：`educationStage`
**类型**：`select`
**必填**：是

**说明**：
标识是哪个教育阶段的同学，用于分类和快速检索。

**选项定义**：

| 值 | 中文 | 英文 | 说明 | 年龄段 |
|----|------|------|------|--------|
| primary | 小学 | Primary School | 小学阶段 | 6-12岁 |
| middle | 初中 | Middle School | 初中阶段 | 12-15岁 |
| high | 高中 | High School | 高中阶段 | 15-18岁 |
| vocational | 中专/职高 | Vocational School | 职业教育 | 15-18岁 |
| college | 大专 | College | 专科教育 | 18-21岁 |
| university | 大学本科 | University | 本科教育 | 18-22岁 |
| master | 硕士研究生 | Master's | 硕士研究生 | 22-25岁 |
| phd | 博士研究生 | PhD | 博士研究生 | 25-28岁+ |
| training | 培训班/进修 | Training Program | 各类培训课程 | 任意年龄 |

**使用场景**：
- 按教育阶段筛选同学
- 组织同学聚会（小学同学会、大学同学会）
- 统计分析（各阶段同学数量）

---

### 3.3 年级班级（gradeClass）

**字段名**：`gradeClass`
**类型**：`string`
**必填**：否

**说明**：
记录更细致的年级和班级信息，用于同学细分。

**示例值**：
- "2015级"
- "三年二班"
- "高三(5)班"
- "计算机科学1班"
- "Class of 2018"
- "MBA 2023秋季班"

**格式建议**：
- 中小学：年级+班级（如：三年二班）
- 大学：届别+专业+班级（如：2015级计算机1班）
- 研究生：年份+专业/方向（如：2020级人工智能方向）
- 培训班：期数+班次（如：第10期周末班）

**验证规则**：
- 最大长度：50字符

---

### 3.4 同学类型（classmateType）

**字段名**：`classmateType`
**类型**：`select`
**必填**：是

**说明**：
区分同学关系的亲疏程度，影响关系强度和交往频率。

**选项定义**：

| 值 | 中文 | 英文 | 说明 | 特点 |
|----|------|------|------|------|
| same_class | 同班同学 | Same Class | 同一个班级 | 最熟悉，接触最多 |
| same_grade | 同年级（不同班） | Same Grade | 同年级不同班 | 认识但不太熟 |
| same_school | 同校（不同年级） | Same School | 同校不同年级 | 学长学姐、学弟学妹 |
| dorm_mate | 室友/舍友 | Dorm Mate | 住同一宿舍 | 关系通常很好 |
| club_mate | 社团/兴趣班 | Club Mate | 同一社团或兴趣小组 | 有共同兴趣 |

**默认值**：`same_class`（最常见）

**关系强度影响**：
- same_class > dorm_mate > club_mate > same_grade > same_school

---

### 3.5 共同活动（activities）

**字段名**：`activities`
**类型**：`multiSelect`（多选）
**必填**：否

**说明**：
记录在学校期间共同参与过的活动，有助于回忆和建立话题。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| study_group | 学习小组 | Study Group | 一起学习、做作业 |
| sports_team | 体育队 | Sports Team | 篮球队、足球队等 |
| club | 社团活动 | Club Activities | 学生会、文学社、动漫社等 |
| competition | 竞赛/比赛 | Competition | 数学竞赛、辩论赛等 |
| event | 校园活动 | School Event | 运动会、文艺演出等 |
| volunteer | 志愿活动 | Volunteer | 志愿者、公益活动 |

**使用示例**：
```javascript
activities: ["study_group", "sports_team", "club"]
// 表示：一起学习、打球、参加社团
```

**使用价值**：
- 回忆共同经历
- 找到共同话题
- 组织主题聚会（如：篮球队聚会）

---

### 3.6 毕业后联系状态（postGradContact）

**字段名**：`postGradContact`
**类型**：`select`
**必填**：否

**说明**：
记录毕业后的联系情况，帮助维护同学关系。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| frequent | 经常联系 | Frequent | 毕业后仍保持频繁联系 |
| occasional | 偶尔联系 | Occasional | 偶尔聊天，保持联系 |
| reunion_only | 仅聚会时 | Reunion Only | 只在同学聚会时见面 |
| lost_contact | 已失联 | Lost Contact | 毕业后失去联系 |

**默认值**：`occasional`

**使用场景**：
- 筛选活跃同学（组织聚会邀请名单）
- 识别失联同学（考虑重新联系）
- 统计毕业后的社交维护情况

**智能提醒**：
- 如果是"同班同学"但"已失联"，系统提示"要不要重新联系一下？"

## 4. 配置说明

### 4.1 时间段配置

```javascript
supportMultiPeriod: true,
requirePeriod: true
```

**理由**：
- 求学时间通常很明确（入学-毕业）
- 可能在多个学校与同一人是同学（如：初中、高中都在同一学校）
- 时间段有助于追溯教育历程

**典型场景**：
```javascript
// 场景1：初中和高中都是同学
periods: [
  {
    startTime: "2005-09-01",
    endTime: "2008-06-30",
    note: "初中同学"
  },
  {
    startTime: "2008-09-01",
    endTime: "2011-06-30",
    note: "高中同学"
  }
]

// 场景2：大学四年
periods: [
  {
    startTime: "2015-09-01",
    endTime: "2019-06-30"
  }
]

// 场景3：在读研究生（还未毕业）
periods: [
  {
    startTime: "2023-09-01",
    endTime: null
  }
]
```

---

### 4.2 关系方向配置

```javascript
bidirectional: true
```

**说明**：
- 同学关系是完全对等的双向关系
- A是B的同学，B也是A的同学
- 属性完全一致，无需区分方向

---

### 4.3 优先级设置

```javascript
priority: 68
```

**理由**：
- 同学关系在社交网络中很重要，特别是关系好的同学
- 优先级68：高于普通同事（65）
- 低于亲密好友（70）和家庭关系（80+）
- 体现了同学在人生中的重要位置

**考虑因素**：
- 同学关系往往能持续一生
- 是重要的社交网络和人脉资源
- 有共同的回忆和经历

## 6. 实现代码

### 6.1 类型类实现

```javascript
// types/education_classmate.type.js
const BaseGuanxiType = require('./base.type');

/**
 * 同学关系类型
 */
class EducationClassmateType extends BaseGuanxiType {
  constructor() {
    super({
      _id: 'education_classmate',
      name: '同学关系',
      nameEn: 'Classmate',
      nameI18nKey: 'guanxiType.education_classmate.name',
      icon: 'book',
      color: '#A8E6CF',
      category: 'education',
      description: '小学、中学、大学等各教育阶段的同学关系',
      descriptionEn: 'Classmates from primary school, secondary school, university, and other educational stages',

      fields: [
        // ... (完整字段定义见第2节)
      ],

      config: {
        supportMultiPeriod: true,
        requirePeriod: true,
        bidirectional: true,
        showInGraph: true,
        priority: 68,
        allowDuplicate: false,
        autoReminder: false
      }
    });
  }

  /**
   * 计算关系强度
   * 同学关系的强度由同学类型、共同活动、毕业后联系状态共同决定
   */
  calculateStrength(attributes) {
    let strength = 45; // 基础分（比一般社交关系高）

    // 根据同学类型加分
    const typeScores = {
      same_class: 30,     // 同班关系最强
      dorm_mate: 28,      // 室友关系很好
      club_mate: 22,      // 社团伙伴
      same_grade: 18,     // 同年级一般
      same_school: 12     // 同校较弱
    };
    if (attributes.classmateType) {
      strength += typeScores[attributes.classmateType] || 0;
    }

    // 根据共同活动加分
    if (attributes.activities && attributes.activities.length > 0) {
      strength += Math.min(attributes.activities.length * 5, 20);
    }

    // 根据毕业后联系状态调整
    const contactMultipliers = {
      frequent: 1.2,         // 经常联系，增加20%
      occasional: 1.0,       // 偶尔联系，保持不变
      reunion_only: 0.8,     // 仅聚会，降低20%
      lost_contact: 0.5      // 失联，降低50%
    };
    if (attributes.postGradContact) {
      strength *= contactMultipliers[attributes.postGradContact] || 1.0;
    }

    // 教育阶段越早，感情基础可能越深
    const stageBonus = {
      primary: 5,      // 小学同学感情深
      middle: 4,
      high: 3,
      university: 2,
      master: 1,
      phd: 1
    };
    if (attributes.educationStage) {
      strength += stageBonus[attributes.educationStage] || 0;
    }

    return Math.min(Math.round(strength), 100);
  }

  /**
   * 生成关系描述文本
   */
  generateDescription(attributes) {
    const parts = [];

    // 学校和教育阶段
    if (attributes.school) {
      let schoolInfo = attributes.school;

      const stageLabels = {
        primary: '小学',
        middle: '初中',
        high: '高中',
        vocational: '中专',
        college: '大专',
        university: '大学',
        master: '硕士',
        phd: '博士',
        training: '培训班'
      };
      if (attributes.educationStage) {
        schoolInfo += ` (${stageLabels[attributes.educationStage]})`;
      }

      parts.push(schoolInfo);
    }

    // 年级班级
    if (attributes.gradeClass) {
      parts.push(attributes.gradeClass);
    }

    // 同学类型
    const typeLabels = {
      same_class: '同班',
      dorm_mate: '室友',
      club_mate: '社团',
      same_grade: '同级',
      same_school: '同校'
    };
    if (attributes.classmateType) {
      parts.push(typeLabels[attributes.classmateType]);
    }

    return parts.join(' ');
  }

  /**
   * 验证属性值
   */
  validate(attributes) {
    const errors = [];

    // 必填验证
    if (!attributes.school || attributes.school.trim() === '') {
      errors.push({
        field: 'school',
        message: '学校名称不能为空'
      });
    }

    if (!attributes.educationStage) {
      errors.push({
        field: 'educationStage',
        message: '请选择教育阶段'
      });
    }

    if (!attributes.classmateType) {
      errors.push({
        field: 'classmateType',
        message: '请选择同学类型'
      });
    }

    // 长度验证
    if (attributes.school && attributes.school.length > 100) {
      errors.push({
        field: 'school',
        message: '学校名称不能超过100个字符'
      });
    }

    if (attributes.gradeClass && attributes.gradeClass.length > 50) {
      errors.push({
        field: 'gradeClass',
        message: '年级班级不能超过50个字符'
      });
    }

    return {
      valid: errors.length === 0,
      errors
    };
  }

  /**
   * 智能建议
   * 根据已有信息提供智能建议
   */
  getSuggestions(attributes) {
    const suggestions = [];

    // 如果是同班同学但失联，建议重新联系
    if (attributes.classmateType === 'same_class' &&
        attributes.postGradContact === 'lost_contact') {
      suggestions.push({
        type: 'reconnect',
        message: '同班同学失联了，要不要尝试重新联系？',
        action: 'create_reminder'
      });
    }

    // 如果有很多共同活动但来往不多，建议加强联系
    if (attributes.activities && attributes.activities.length >= 3 &&
        attributes.postGradContact === 'reunion_only') {
      suggestions.push({
        type: 'strengthen',
        message: '你们有很多共同经历，可以多联系聊聊',
        action: 'suggest_topics'
      });
    }

    return suggestions;
  }
}

module.exports = EducationClassmateType;
```

## 8. 验证规则

### 8.1 时间段有效性验证

```javascript
function validateEducationPeriod(period, educationStage) {
  const errors = [];

  // 计算持续时长
  const start = new Date(period.startTime);
  const end = period.endTime ? new Date(period.endTime) : new Date();
  const years = (end - start) / (365.25 * 24 * 3600 * 1000);

  // 不同教育阶段的合理年限
  const expectedDurations = {
    primary: { min: 3, max: 7 },      // 小学通常6年，允许±1
    middle: { min: 2, max: 4 },       // 初中通常3年
    high: { min: 2, max: 4 },         // 高中通常3年
    vocational: { min: 2, max: 4 },   // 中专通常3年
    college: { min: 2, max: 4 },      // 大专通常3年
    university: { min: 3, max: 6 },   // 本科通常4年
    master: { min: 2, max: 4 },       // 硕士2-3年
    phd: { min: 3, max: 8 },          // 博士3-6年
    training: { min: 0, max: 5 }      // 培训班灵活
  };

  const expected = expectedDurations[educationStage];
  if (expected && (years < expected.min || years > expected.max)) {
    errors.push({
      type: 'warning',
      message: `${educationStage}阶段通常为${expected.min}-${expected.max}年，当前为${years.toFixed(1)}年，请确认时间是否正确`
    });
  }

  return errors;
}
```

### 8.2 业务逻辑验证

```javascript
function validateBusinessLogic(attributes, periods) {
  const warnings = [];

  // 如果是室友但不是同年级，给出提示
  if (attributes.classmateType === 'dorm_mate' &&
      attributes.educationStage &&
      !['university', 'college', 'master', 'phd'].includes(attributes.educationStage)) {
    warnings.push({
      type: 'info',
      message: '小学/中学通常不住校，确认是否为室友关系？'
    });
  }

  // 培训班通常时间较短
  if (attributes.educationStage === 'training' &&
      periods && periods[0]) {
    const start = new Date(periods[0].startTime);
    const end = periods[0].endTime ? new Date(periods[0].endTime) : new Date();
    const months = (end - start) / (30 * 24 * 3600 * 1000);

    if (months > 24) {
      warnings.push({
        type: 'warning',
        message: '培训班时间超过2年，请确认教育阶段是否选择正确'
      });
    }
  }

  // 同班但失联，建议重联
  if (attributes.classmateType === 'same_class' &&
      attributes.postGradContact === 'lost_contact') {
    warnings.push({
      type: 'suggestion',
      message: '同班同学失联了，可以通过校友录或社交媒体尝试重新联系'
    });
  }

  return warnings;
}
```
