## 1. 类型概述

### 1.1 业务场景

好友关系是最常见的社交关系类型，涵盖了各种朋友关系。与亲属关系不同，好友关系可能会中断、恢复，或随时间变化。

**典型使用场景**：
- 记录朋友的基本信息和关系历史
- 管理不同类型的朋友（闺蜜、发小、网友等）
- 追踪友情的发展历程
- 维护社交圈层

### 1.2 设计目标

1. **记录关系建立**：记录认识时间、场合
2. **区分亲疏**：通过亲密度、来往频率区分好友层次
3. **支持变化**：支持多时间段，记录友情的起伏
4. **灵活标签**：通过共同兴趣等标签丰富关系描述

### 1.3 关键特点

- ✓ 支持多时间段（友情可能中断后恢复）
- ✓ 不要求必填时间段（有些老朋友记不清认识时间）
- ✓ 双向关系（互为朋友）
- ✓ 中高优先级（优先级：70）
- ✓ 6个专属字段：好友类型、认识场合、亲密程度、来往频率、共同兴趣、友情状态

## 3. 字段详细说明

### 3.1 好友类型（friendType）

**字段名**：`friendType`
**类型**：`select`
**必填**：是

**说明**：
标识朋友关系的类型，用于快速分类和识别。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| best_friend | 闺蜜/死党 | Best Friend | 最亲密的朋友，无话不谈 |
| childhood | 发小/青梅竹马 | Childhood Friend | 从小一起长大的朋友 |
| close | 好朋友 | Close Friend | 关系很好，经常联系的朋友 |
| casual | 普通朋友 | Casual Friend | 认识但不太深入的朋友 |
| online | 网友 | Online Friend | 通过网络认识的朋友 |
| acquaintance | 泛泛之交 | Acquaintance | 认识但交往不多 |

**默认值**：`close`

**使用建议**：
- 根据实际关系选择最贴切的类型
- 闺蜜/死党：可以分享秘密的最好朋友
- 发小：从小认识，有深厚感情基础
- 网友：线上认识，可能没见过面

---

### 3.2 认识场合（metAt）

**字段名**：`metAt`
**类型**：`string`
**必填**：否

**说明**：
记录在哪里认识的这位朋友，帮助回忆友情的起点。

**示例值**：
- "大学宿舍"
- "健身房"
- "旅行途中"
- "朋友介绍"
- "网络论坛"
- "公司年会"
- "兴趣小组"

**验证规则**：
- 最大长度：50字符

**使用建议**：
- 尽量简洁明了
- 可以包含时间信息（如"2019年云南旅行"）
- 如果记不清可以留空

---

### 3.3 亲密程度（intimacy）

**字段名**：`intimacy`
**类型**：`select`
**必填**：否

**说明**：
描述当前友情的亲密程度，用于关系强度计算和好友分层。

**选项定义**：

| 值 | 中文 | 英文 | 说明 | 特征 |
|----|------|------|------|------|
| very_close | 非常亲密 | Very Close | 无话不谈的挚友 | 分享秘密、互相支持、深度信任 |
| close | 比较亲密 | Close | 关系很好的朋友 | 经常聊天、互相帮忙、互相关心 |
| moderate | 一般 | Moderate | 普通朋友关系 | 偶尔联系、浅层交往 |
| distant | 疏远 | Distant | 关系变淡的朋友 | 很少联系、渐行渐远 |

**默认值**：`moderate`

**与好友类型的关系**：
- 好友类型：关系的定位（闺蜜、发小等）
- 亲密程度：当前的友情质量
- 例如：类型是"闺蜜"但目前"疏远"（曾经很好但现在渐远）

---

### 3.4 来往频率（contactFrequency）

**字段名**：`contactFrequency`
**类型**：`select`
**必填**：否

**说明**：
记录与该朋友的实际来往频率，用于关系维护提醒。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| daily | 每天 | Daily | 几乎每天都聊天或见面 |
| weekly | 每周 | Weekly | 每周至少联系一次 |
| monthly | 每月 | Monthly | 每月至少联系一次 |
| quarterly | 每季度 | Quarterly | 每3个月左右联系一次 |
| yearly | 每年 | Yearly | 通常每年见一两次 |
| rarely | 很少联系 | Rarely | 很少主动联系 |

**智能提醒**：
- 如果是"非常亲密"但"很少联系"，系统可以提醒"是不是该联系一下XX了？"
- 根据历史来往记录预测下次该联系的时间

### 3.5 共同兴趣（commonInterests）

**字段名**：`commonInterests`
**类型**：`multiSelect`（多选）

**说明**：
记录你们之间的共同兴趣爱好，可以选择多个。有助于：
- 找话题（聊共同兴趣）
- 组织活动（邀请有共同兴趣的朋友）
- 朋友分组（按兴趣分类）

**选项定义**：

| 值 | 中文 | 英文 |
|----|------|------|
| sports | 运动健身 | Sports & Fitness |
| music | 音乐 | Music |
| movies | 电影 | Movies |
| reading | 阅读 | Reading |
| travel | 旅行 | Travel |
| gaming | 游戏 | Gaming |
| food | 美食 | Food |
| photography | 摄影 | Photography |
| tech | 科技数码 | Technology |

**使用示例**：
```javascript

// 表示：喜欢一起运动、旅行、吃美食
```

### 3.6 友情状态（friendshipStatus）

**字段名**：`friendshipStatus`
**类型**：`select`
**必填**：否

**说明**：
记录当前友情的状态，特别是对于经历起伏的友情。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| active | 正常往来 | Active | 友情正常，保持联系 |
| fading | 逐渐疏远 | Fading | 感觉关系在变淡 |
| estranged | 失联 | Estranged | 已经失去联系 |
| reconnected | 重新联系 | Reconnected | 曾失联但最近重新联系上了 |

**默认值**：`active`

**使用场景**：
- 状态为"fading"时，系统提示"要不要主动联系一下？"
- 状态为"estranged"时，如果重新联系，更新为"reconnected"并添加新时间段
- 状态为"reconnected"时，可以在时间线中标记重逢事件

## 4. 配置说明

### 4.1 多时间段支持

```javascript
supportMultiPeriod: true,
requirePeriod: false
```

**理由**：
- 友情可能会经历中断和恢复
- 例如：大学时是好友（2010-2014），毕业后失联，2020年又重新联系上
- 支持多时间段可以完整记录友情历程
- 但不强制要求填写时间段（很多老朋友记不清确切时间）

**数据示例**：
```javascript
periods: [
  { startTime: "2010-09-01", endTime: "2014-06-30" },  // 大学期间
  { startTime: "2020-03-15", endTime: null }           // 重新联系后至今
]
```

---

### 4.2 自动提醒配置

```javascript
autoReminder: true,
reminderRules: [
  {
    type: "birthday",
    advance: 1,
    condition: {
      field: "intimacy",
      value: ["very_close", "close"]
    }
  }
]
```

**说明**：
- 自动为亲密好友创建生日提醒
- 提前1天提醒，避免错过好友生日
- 仅对"非常亲密"和"比较亲密"的好友创建提醒
- 用户可以手动关闭或修改提醒

---

### 4.3 优先级设置

```javascript
priority: 70
```

**理由**：
- 好友关系在社交关系中非常重要
- 优先级70：高于普通同事（60）、邻居（40）
- 低于家庭关系（80-100）
- 在图谱中会用较粗的线和较大的节点显示

## 6. 实现代码

### 6.1 类型类实现

```javascript
// types/social_friend.type.js
const BaseGuanxiType = require('./base.type');

/**
 * 好友关系类型
 */
class SocialFriendType extends BaseGuanxiType {
  constructor() {
    super({
      _id: 'social_friend',
      name: '好友关系',
      nameEn: 'Friend',
      nameI18nKey: 'guanxiType.social_friend.name',
      icon: 'smile',
      color: '#4ECDC4',
      category: 'social',
      description: '朋友、闺蜜、发小、网友等社交关系',
      descriptionEn: 'Friends, best friends, childhood friends, online friends, and other social relationships',

      fields: [
        // ... (完整字段定义见第2节)
      ],

      config: {
        supportMultiPeriod: true,
        requirePeriod: false,
        bidirectional: true,
        showInGraph: true,
        priority: 70,
        allowDuplicate: false,
        autoReminder: true,
        reminderRules: [
          {
            type: 'birthday',
            advance: 1,
            condition: {
              field: 'intimacy',
              value: ['very_close', 'close']
            }
          }
        ]
      }
    });
  }

  /**
   * 计算关系强度
   * 好友关系的强度由好友类型、亲密程度、来往频率共同决定
   */
  calculateStrength(attributes) {
    let strength = 40; // 基础分

    // 根据好友类型加分
    const typeScores = {
      best_friend: 40,
      childhood: 35,
      close: 30,
      casual: 20,
      online: 15,
      acquaintance: 10
    };
    if (attributes.friendType) {
      strength += typeScores[attributes.friendType] || 0;
    }

    // 根据亲密程度加分
    const intimacyScores = {
      very_close: 30,
      close: 20,
      moderate: 10,
      distant: 0
    };
    if (attributes.intimacy) {
      strength += intimacyScores[attributes.intimacy] || 0;
    }

    // 根据来往频率加分
    const frequencyScores = {
      daily: 20,
      weekly: 15,
      monthly: 10,
      quarterly: 5,
      yearly: 2,
      rarely: 0
    };
    if (attributes.contactFrequency) {
      strength += frequencyScores[attributes.contactFrequency] || 0;
    }

    // 友情状态调整
    if (attributes.friendshipStatus === 'fading') {
      strength *= 0.8;  // 疏远中，降低20%
    } else if (attributes.friendshipStatus === 'estranged') {
      strength *= 0.5;  // 失联，降低50%
    } else if (attributes.friendshipStatus === 'reconnected') {
      strength *= 0.9;  // 重联，略降10%（需要时间恢复）
    }

    return Math.min(Math.round(strength), 100);
  }

  /**
   * 生成关系描述文本
   */
  generateDescription(attributes) {
    const parts = [];

    // 好友类型
    const typeLabels = {
      best_friend: '闺蜜',
      childhood: '发小',
      close: '好友',
      casual: '朋友',
      online: '网友',
      acquaintance: '熟人'
    };
    if (attributes.friendType) {
      parts.push(typeLabels[attributes.friendType]);
    }

    // 认识场合
    if (attributes.metAt) {
      parts.push(`(${attributes.metAt}认识)`);
    }

    // 共同兴趣
    if (attributes.commonInterests && attributes.commonInterests.length > 0) {
      const interestLabels = {
        sports: '运动',
        music: '音乐',
        movies: '电影',
        reading: '阅读',
        travel: '旅行',
        gaming: '游戏',
        food: '美食',
        photography: '摄影',
        tech: '科技',
        other: '其他'
      };
      const interests = attributes.commonInterests
        .slice(0, 3)
        .map(i => interestLabels[i])
        .join('、');
      parts.push(`共同兴趣：${interests}`);
    }

    return parts.join(' ');
  }

  /**
   * 验证属性值
   */
  validate(attributes) {
    const errors = [];

    // 必填验证
    if (!attributes.friendType) {
      errors.push({
        field: 'friendType',
        message: '请选择好友类型'
      });
    }

    // 长度验证
    if (attributes.metAt && attributes.metAt.length > 50) {
      errors.push({
        field: 'metAt',
        message: '认识场合不能超过50个字符'
      });
    }

    // 枚举验证
    const validFriendTypes = ['best_friend', 'childhood', 'close', 'casual', 'online', 'acquaintance'];
    if (attributes.friendType && !validFriendTypes.includes(attributes.friendType)) {
      errors.push({
        field: 'friendType',
        message: '好友类型值无效'
      });
    }

    return {
      valid: errors.length === 0,
      errors
    };
  }

  /**
   * 自动提醒规则
   */
  shouldCreateReminder(attributes, reminderType) {
    if (reminderType === 'birthday') {
      // 只为亲密好友创建生日提醒
      return attributes.intimacy === 'very_close' ||
             attributes.intimacy === 'close' ||
             attributes.friendType === 'best_friend';
    }
    return false;
  }
}

module.exports = SocialFriendType;
```

function validateCombination(attributes) {
  const warnings = [];

  // 亲密程度与来往频率不匹配时给出提示
  if (attributes.intimacy === 'very_close' &&
      attributes.contactFrequency === 'rarely') {
    warnings.push({
      type: 'warning',
      message: '非常亲密的朋友但很少联系，是不是该主动联系一下了？'
    });
  }

  // 友情状态与来往频率不匹配
  if (attributes.friendshipStatus === 'active' &&
      attributes.contactFrequency === 'rarely') {
    warnings.push({
      type: 'warning',
      message: '状态为正常往来但很少联系，建议更新友情状态或增加联系频率'
    });
  }

  return warnings;
}
