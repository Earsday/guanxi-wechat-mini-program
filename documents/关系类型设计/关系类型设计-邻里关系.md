## 1. 类型概述

### 1.1 业务场景

邻里关系是基于居住地邻近而建立的社交关系。好的邻里关系能提升居住幸福感，在日常生活中互相帮助。

**典型使用场景**：
- 记录小区邻居信息
- 管理邻里互助关系
- 社区活动组织
- 搬家后维护老邻居关系

### 1.2 设计目标

1. **明确位置信息**：小区、楼栋、单元等地理位置
2. **区分远近**：同楼层、同楼栋、同小区
3. **时间段管理**：搬家会导致邻里关系中断
4. **互助记录**：记录邻里互助情况

### 1.3 关键特点

- ✓ 支持多时间段（搬家会中断邻里关系）
- ✓ 要求填写时间段（居住时间通常明确）
- ✓ 双向关系（互为邻居）
- ✓ 中等优先级（优先级：40）
- ✓ 5个专属字段：小区名称、楼栋单元、邻近程度、互助关系、社区活动

## 3. 字段详细说明

### 3.1 小区名称（community）

**字段名**：`community`
**类型**：`string`
**必填**：是

**说明**：
记录居住的小区或社区名称，是邻里关系的核心标识。

**示例值**：
- "万科城市花园"
- "碧桂园凤凰城"
- "阳光小区"
- "XX路XX号院"
- "Greenwood Apartments"（国外公寓）

**验证规则**：
- 最大长度：100字符
- 不能为空

**使用建议**：
- 使用小区/社区的正式名称
- 如果是老式住宅区，可以用街道+门牌号
- 国外可以用公寓或街区名称

---

### 3.2 楼栋单元（building）

**字段名**：`building`
**类型**：`string`
**必填**：否

**说明**：
记录更具体的楼栋、单元、楼层信息，帮助定位邻居的位置。

**示例值**：
- "3号楼2单元"
- "A栋5层"
- "东区15号楼"
- "Building A, Unit 502"

**格式建议**：
- 楼栋+单元：3号楼2单元
- 楼栋+楼层：A栋5层
- 门牌号：502室

**验证规则**：
- 最大长度：50字符

**使用场景**：
- 定位邻居具体位置
- 按楼栋筛选邻居
- 组织楼栋活动

---

### 3.3 邻近程度（proximity）

**字段名**：`proximity`
**类型**：`select`
**必填**：是

**说明**：
标识邻居的远近程度，影响来往频率和关系强度。

**选项定义**：

| 值 | 中文 | 英文 | 说明 | 特点 |
|----|------|------|------|------|
| same_floor | 同楼层 | Same Floor | 同一层楼 | 经常碰面，关系最近 |
| same_building | 同楼栋 | Same Building | 同一栋楼 | 经常在电梯、楼道遇见 |
| same_community | 同小区 | Same Community | 同一小区 | 偶尔在小区内遇见 |
| nearby | 附近小区 | Nearby | 相邻小区或附近社区 | 很少偶遇，需要特意联系 |

**默认值**：`same_community`（最常见）

**关系强度影响**：
- same_floor（最强）> same_building > same_community > nearby（最弱）

**使用场景**：
- 计算关系强度（越近关系通常越好）
- 筛选最近的邻居（紧急求助时）
- 图谱可视化（用距离表示邻近程度）

---

### 3.4 互助关系（mutualHelp）

**字段名**：`mutualHelp`
**类型**：`multiSelect`（多选）
**必填**：否

**说明**：
记录与邻居之间的互助情况，体现邻里关系的深度。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| emergency | 紧急互助 | Emergency Help | 紧急情况可以互相帮助 |
| keys | 互留钥匙 | Key Exchange | 互相保管备用钥匙 |
| packages | 代收快递 | Package Receipt | 帮忙代收快递包裹 |
| pets | 照看宠物 | Pet Care | 互相照看宠物 |
| tools | 工具借用 | Tool Sharing | 借用工具、物品 |
| carpooling | 拼车/顺路 | Carpooling | 上下班顺路、拼车 |

**使用示例**：
```javascript
mutualHelp: ["packages", "keys", "emergency"]
// 表示：可以代收快递、互留钥匙、紧急情况互助
```

**关系强度影响**：
- 互助类型越多，关系越好
- 特别是"互留钥匙"和"紧急互助"，说明信任度很高

---

### 3.5 社区活动（communityActivities）

**字段名**：`communityActivities`
**类型**：`multiSelect`（多选）
**必填**：否

**说明**：
记录与邻居的日常交往活动，体现邻里关系的质量。

**选项定义**：

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| exercise | 一起锻炼 | Exercise Together | 一起晨练、散步、跳广场舞等 |
| grocery | 一起买菜 | Grocery Shopping | 结伴去菜市场、超市 |
| kids_play | 孩子玩伴 | Kids Playdate | 孩子们一起玩 |
| chat | 闲聊 | Casual Chat | 经常聊天、拉家常 |
| events | 小区活动 | Community Events | 参加小区组织的活动 |

**使用示例**：
```javascript
communityActivities: ["exercise", "grocery", "chat"]
// 表示：经常一起晨练、买菜、聊天
```

**使用价值**：
- 体现邻里关系的活跃度
- 找到共同兴趣的邻居
- 组织社区活动时的参考

## 4. 配置说明

### 4.1 时间段配置

```javascript
supportMultiPeriod: true,
requirePeriod: true
```

**理由**：
- 搬家会导致邻里关系中断
- 居住时间通常很明确（租房合同、购房时间）
- 可能搬回原小区，与老邻居重新成为邻居

**典型场景**：
```javascript
// 场景1：当前居住
periods: [
  {
    startTime: "2020-06-01",
    endTime: null,
    note: "现居小区"
  }
]

// 场景2：已搬走
periods: [
  {
    startTime: "2015-03-01",
    endTime: "2020-05-31",
    note: "2020年搬家"
  }
]

// 场景3：多次搬家后又回到同一小区
periods: [
  {
    startTime: "2010-01-01",
    endTime: "2015-12-31"
  },
  {
    startTime: "2022-06-01",
    endTime: null,
    note: "搬回来了"
  }
]
```

---

### 4.2 关系方向配置

```javascript
bidirectional: true
```

**说明**：
- 邻里关系是完全对等的双向关系
- A是B的邻居，B也是A的邻居
- 属性完全一致

---

### 4.3 优先级设置

```javascript
priority: 40
```

**理由**：
- 邻里关系在人际网络中是辅助性的
- 优先级40：低于工作（65）、同学（68）、好友（70）
- 高于普通社交关系（30）
- 体现了邻里关系的日常生活属性

**注意**：
- 虽然优先级不高，但好的邻里关系对居住体验很重要
- 在"社区视图"中可以提升邻里关系的显示优先级

---

### 4.4 搬家提醒

**功能建议**：
当用户结束某个邻里关系的时间段时（搬家），系统可以：
1. 提醒"是否要保持联系"
2. 建议转为"好友关系"（如果关系很好）
3. 创建"搬家"事件记录

```javascript
        supportMultiPeriod: true,
        requirePeriod: true,
        bidirectional: true,
        showInGraph: true,
        priority: 40,
        allowDuplicate: false,
        autoReminder: false
      }
    });
  }

  /**
   * 计算关系强度
   * 邻里关系的强度由邻近程度、互助关系、社区活动共同决定
   */
  calculateStrength(attributes) {
    let strength = 30; // 基础分（比一般社交关系略低）

    // 根据邻近程度加分
    const proximityScores = {
      same_floor: 30,      // 同楼层关系最强
      same_building: 20,   // 同楼栋经常见面
      same_community: 10,  // 同小区偶尔遇见
      nearby: 5            // 附近小区很少见
    };
    if (attributes.proximity) {
      strength += proximityScores[attributes.proximity] || 0;
    }

    // 根据互助关系加分
    if (attributes.mutualHelp && attributes.mutualHelp.length > 0) {
      const helpScores = {
        emergency: 15,       // 紧急互助说明信任度高
        keys: 12,            // 互留钥匙说明很信任
        packages: 5,         // 代收快递常见
        pets: 8,             // 照看宠物说明关系不错
        tools: 4,            // 工具借用一般
        carpooling: 6        // 拼车说明有交往
      };

      attributes.mutualHelp.forEach(help => {
        strength += helpScores[help] || 0;
      });
    }

    // 根据社区活动加分
    if (attributes.communityActivities && attributes.communityActivities.length > 0) {
      strength += Math.min(attributes.communityActivities.length * 5, 20);
    }

    return Math.min(strength, 100);
  }

  /**
   * 生成关系描述文本
   */
  generateDescription(attributes) {
    const parts = [];

    // 小区和楼栋
    if (attributes.community) {
      let locationInfo = attributes.community;
      if (attributes.building) {
        locationInfo += ` ${attributes.building}`;
      }
      parts.push(locationInfo);
    }

    // 邻近程度
    const proximityLabels = {
      same_floor: '同楼层',
      same_building: '同楼栋',
      same_community: '同小区',
      nearby: '附近'
    };
    if (attributes.proximity) {
      parts.push(`(${proximityLabels[attributes.proximity]})`);
    }

    return parts.join(' ');
  }

  /**
   * 验证属性值
   */
  validate(attributes) {
    const errors = [];

    // 必填验证
    if (!attributes.community || attributes.community.trim() === '') {
      errors.push({
        field: 'community',
        message: '小区名称不能为空'
      });
    }

    if (!attributes.proximity) {
      errors.push({
        field: 'proximity',
        message: '请选择邻近程度'
      });
    }

    // 长度验证
    if (attributes.community && attributes.community.length > 100) {
      errors.push({
        field: 'community',
        message: '小区名称不能超过100个字符'
      });
    }

    if (attributes.building && attributes.building.length > 50) {
      errors.push({
        field: 'building',
        message: '楼栋单元不能超过50个字符'
      });
    }

    return {
      valid: errors.length === 0,
      errors
    };
  }

  /**
   * 智能建议
   */
  getSuggestions(attributes) {
    const suggestions = [];

    // 如果是同楼层但没有互助关系，建议建立互助
    if (attributes.proximity === 'same_floor' &&
        (!attributes.mutualHelp || attributes.mutualHelp.length === 0)) {
      suggestions.push({
        type: 'improve',
        message: '同楼层邻居，可以考虑建立互助关系（如代收快递）',
        action: 'suggest_mutual_help'
      });
    }

    // 如果互留钥匙但没有紧急互助，建议添加
    if (attributes.mutualHelp?.includes('keys') &&
        !attributes.mutualHelp.includes('emergency')) {
      suggestions.push({
        type: 'complete',
        message: '既然互留钥匙，可以标记"紧急互助"',
        action: 'add_emergency_help'
      });
    }

    return suggestions;
  }

  /**
   * 搬家检测
   * 检测是否需要结束邻里关系
   */
  checkMoving(currentAddress, neighborAddress) {
    // 如果地址不再邻近，提示可能已搬家
    if (!this.isAddressNearby(currentAddress, neighborAddress)) {
      return {
        suggestAction: 'end_period',
        message: '地址已不邻近，是否已搬家？是否结束邻里关系时间段？'
      };
    }
    return null;
  }
}

module.exports = NeighborCommunityType;
```

## 8. 验证规则

### 8.1 时间段合理性验证

```javascript
function validateResidencePeriod(period) {
  const warnings = [];

  // 计算居住时长
  const start = new Date(period.startTime);
  const end = period.endTime ? new Date(period.endTime) : new Date();
  const years = (end - start) / (365.25 * 24 * 3600 * 1000);

  // 居住时间异常检测
  if (years < 0.1) {
    warnings.push({
      type: 'warning',
      message: '居住时间少于1个月，是否时间填写有误？'
    });
  }

  if (years > 50) {
    warnings.push({
      type: 'warning',
      message: '居住时间超过50年，请确认时间是否正确'
    });
  }

  return warnings;
}
```

### 8.2 业务逻辑验证

```javascript
function validateBusinessLogic(attributes) {
  const warnings = [];

  // 附近小区但互留钥匙，逻辑不合理
  if (attributes.proximity === 'nearby' &&
      attributes.mutualHelp?.includes('keys')) {
    warnings.push({
      type: 'warning',
      message: '附近小区通常不会互留钥匙，请确认邻近程度或互助关系是否正确'
    });
  }

  // 同楼层但没有任何社区活动，建议加强交往
  if (attributes.proximity === 'same_floor' &&
      (!attributes.communityActivities || attributes.communityActivities.length === 0) &&
      (!attributes.mutualHelp || attributes.mutualHelp.length === 0)) {
    warnings.push({
      type: 'suggestion',
      message: '同楼层邻居，建议建立更多互动（如代收快递、偶尔闲聊）'
    });
  }

  return warnings;
}
```
