#### 4.2.4 创建关系（guanxi/create）

**交互流程**：
1. 选择关系双方（人物选择器）
2. 选择关系类型
3. 填写关系专属属性（动态表单）
4. 设置时间段
5. 设置关系强度和备注
6. 确认创建

**动态表单**：
根据选择的关系类型，动态渲染对应的属性字段。

```javascript
// 示例：根据类型生成表单
function generateForm(typeId) {
  const type = getTypeById(typeId);
  return type.fields.map(field => {
    switch (field.type) {
      case 'string':
        return `<input type="text" name="${field.name}" />`;
      case 'select':
        return `<select name="${field.name}">
          ${field.options.map(opt => `<option>${opt}</option>`).join('')}
        </select>`;
      // ... 其他类型
    }
  });
}
```

**布局**：

**功能**：

- 布局切换（力导向、环形、网格）
- 节点点击查看详情
- 节点拖动
- 缩放和平移
- 路径查找（点击两个节点显示路径）

**技术实现**：
使用 AntV F6 或 ECharts Graph 组件。

```javascript
// 示例：使用 F6
import F6 from '@antv/f6-wx';

Page({
  data: {
    graphData: {
      nodes: [],
      edges: []
    }
  },

  onLoad() {
    this.initGraph();
    this.loadGraphData();
  },

  initGraph() {
    const ctx = wx.createCanvasContext('graph-canvas');
    this.graph = new F6.Graph({
      context: ctx,
      width: 750,
      height: 600,
      layout: {
        type: 'force',
        preventOverlap: true
      },
      defaultNode: {
        size: 30,
        style: {
          fill: '#5B8FF9',
          stroke: '#fff',
          lineWidth: 2
        }
      },
      defaultEdge: {
        style: {
          stroke: '#e2e2e2'
        }
      }
    });
  },

  async loadGraphData() {
    const res = await wx.cloud.callFunction({
      name: 'graph',
      data: {
        action: 'generateGraph',
        centerCharacterId: this.data.centerCharacterId,
        filters: this.data.filters
      }
    });

    this.graph.data(res.result.data);
    this.graph.render();
  }
});
```

### 4.3 组件设计

#### 4.3.1 人物选择器（character-picker）

**功能**：
- 显示人物列表
- 支持搜索
- 支持多选/单选
- 显示人物头像和姓名

**属性**：
```javascript
{
  mode: 'single' | 'multiple',  // 选择模式
  value: String | Array,         // 已选值
  excludeIds: Array,             // 排除的人物ID
  onChange: Function             // 变化回调
}
```

#### 4.3.2 关系类型选择器（type-selector）

**功能**：
- 显示关系类型列表
- 按分类展示
- 显示图标和名称

#### 4.3.3 时间段编辑器（period-editor）

**功能**：
- 添加/编辑时间段
- 日期选择器
- 开放式时间段（结束时间可为空）
- 时间段备注

**UI**：
```
开始时间： [2010-09-01] [选择]
结束时间： [2015-07-01] [选择] [至今]
备注：     [输入框]
[取消] [确定]
```

#### 4.3.4 动态表单（dynamic-form）

**功能**：
- 根据字段定义动态渲染表单
- 支持多种字段类型
- 自动验证

**实现**：
```javascript
Component({
  properties: {
    fields: Array,     // 字段定义
    values: Object,    // 字段值
    onChange: Function
  },

  methods: {
    renderField(field) {
      switch (field.type) {
        case 'string':
          return this.renderInput(field);
        case 'number':
          return this.renderNumberInput(field);
        case 'date':
          return this.renderDatePicker(field);
        case 'select':
          return this.renderSelect(field);
        case 'multiSelect':
          return this.renderCheckboxGroup(field);
        case 'boolean':
          return this.renderSwitch(field);
      }
    },

    validate() {
      // 验证所有字段
    }
  }
});
```
