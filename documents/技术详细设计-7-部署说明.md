## 7. 部署说明

### 7.1 云函数部署

```bash
# 部署所有云函数
npm run deploy:cloud

# 部署单个云函数
npm run deploy:function character
```

### 7.2 数据库初始化

```javascript
// scripts/init-database.js
// 初始化关系类型
const registry = require('../types/registry');
registry.syncToDatabase(db);

// 创建索引
db.collection('characters').createIndex('userId');
db.collection('characters').createIndex('name', { type: 'text' });
// ...
```

### 7.3 小程序发布

```bash
# 构建
npm run build

# 上传
npm run upload
```
